API Notes
=========

The current backend exposes its application routes under ``/api/v1`` and the health endpoint at ``/health``.

Key route groups
----------------

- ``/api/v1/regions``
- ``/api/v1/triage``
- ``/api/v1/funding``
- ``/api/v1/news``
- ``/api/v1/counterfactual``
- ``/api/v1/charities``
- ``/api/v1/fund``

The OpenAPI schema is served from ``/api/v1/openapi.json`` in the current backend configuration.
